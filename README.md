# The 900

15 minutes is 900 seconds. Taking a moment to breathe and reflect can significantly lower anxiety and increase focus. 

This is a modern, responsive web application designed as a 3-step guided mental wellness session.

## Features:
1. **Journal & Vital Signs Check-in**: A space to free-write your thoughts, along with vertical sliding scales to gauge current Mood and Energy levels.
2. **Environment & Contextual AI**: Select your current environment (Office, Home, Commute, etc.), and receive a personalized 5-minute activity suggestion before starting the breathing exercise.
3. **Apple Watch Style Guided Breathing**: A pure-CSS responsive glowing mandala flower that expands and contracts to guide your breathing over a 5-minute session, complete with Session Pause/End controls.
4. **Session Summary**: A complete summary card detailing the time reclaimed and allowing for a final reflection.

## Technologies Used:
- HTML5
- CSS3 (Vanilla, No Frameworks)
- Vanilla JavaScript (SPA Routing & Timer Logic)

## Setup
Simply clone the repository and open `index.html` in your favorite modern browser. No build steps required.
